Über dieses Projekt

Zielsetzung

This website is designed to test an adaptive menu extension. The goal is to improve user experience by adapting the interface to individual browsing patterns.

Funktionsweise

The browser extension tracks which menu items you click most frequently. Based on this data, it automatically reorganizes the menu to prioritize your most-used pages.

This is inspired by research in adaptive user interfaces and Human-Computer Interaction.

Technologie

  • HTML5 & CSS3 for the website structure
  • JavaScript for the browser extension
  • Chrome Storage API for data persistence
